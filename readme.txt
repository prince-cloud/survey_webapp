TO run this project

1. install python 3.10 or later

2. create a virtuale environemnt using cmd
    - virtualenv "your_virtual_environt"

3. activate your virtal environemnt
    - call "your_virtual_environt\scripts\activate"

4. install requirements
    - python install -r requirements.txt

5. Run migrations
    - python manage.py makemigrations
    - python manage.py migrate

6. Run server
    - python manage.py runserver

7. visit your "localhost:8000"

8. Enjoy!
